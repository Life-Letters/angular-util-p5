# P5.js factory

Quickly use [P5.js](https://p5js.org) in your project.

## Usage

- Add the module `life.util.p5`
- Add the following to your bower file:
	```
	"overrides": {
    "p5.js-sound": {
      "main": [
        "./lib/p5.js",
        "./lib/p5.sound.js"
      ]
    }
  }
	```
- Done

## Development

To run a dev server, use `run.sh`. You will need to have Python installed.
